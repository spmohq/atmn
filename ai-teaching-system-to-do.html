Full Checklist

Phase 2 — Build the Teaching Foundation P2-T01: Generate the Teaching Analysis + Teaching Blueprint. P2-T02: Review the output. P2-T03: Save the Teaching Blueprint. P2-T04: Generate the Mathematics Visual Style DNA. P2-T05: Review the Style DNA. P2-T06: Save the Style DNA for future reuse. Phase 3 — Organize the Knowledge Teacher Mind Map P3-T01: Generate the Teacher Mind Map. P3-T02: Review the Teacher Mind Map. P3-T03: Save the Teacher Mind Map. Student Mind Map P3-T04: Generate the Student Mind Map. P3-T05: Review the Student Mind Map. P3-T06: Save the Student Mind Map. Formula Mind Map P3-T07: Generate the Formula Mind Map. P3-T08: Review the Formula Mind Map. P3-T09: Save the Formula Mind Map. Phase 4 — Generate Classroom Resources P4-T01: Generate Revision Flash Cards. P4-T02: Save the Flash Cards. P4-T03: Generate the Classroom Assessment Quiz. P4-T04: Save the Quiz. P4-T05: Generate the Instructional Infographic. P4-T06: Save the Infographic. P4-T07: Generate the Slide Deck / Presentation. P4-T08: Save the Presentation. Phase 5 — Create Teaching Materials (Gemini) P5-T01: Copy the Teaching Blueprint from NotebookLM. P5-T02: Paste it into Gemini. P5-T03: Generate the complete Lesson Plan. P5-T04: Save the Lesson Plan. P5-T05: Generate the Teaching Strategy for Weak Learners. P5-T06: Save the Weak Learners strategy. P5-T07: Generate the Teaching Strategy for Average Learners. P5-T08: Save the Average Learners strategy. P5-T09: Generate the Teaching Strategy for Advanced Learners. P5-T10: Save the Advanced Learners strategy. P5-T11: Generate the Board Exam Revision Kit. P5-T12: Verify all required sections are included. P5-T13: Save the Revision Kit. P5-T14: Generate Classroom Activities. P5-T15: Save the Classroom Activities. Phase 6 — Create Premium Visuals (Google Flow) Formula Visual P6-T01: Export the Formula Mind Map. P6-T02: Copy the Visual Style DNA. P6-T03: Upload the Formula Mind Map to Google Flow. P6-T04: Paste the Style DNA into the prompt. P6-T05: Generate the Premium Formula Visual. P6-T06: Save the final Formula Visual. Teacher Visual P6-T07: Export the Teacher Mind Map. P6-T08: Upload the Teacher Mind Map to Google Flow. P6-T09: Paste the Style DNA into the prompt. P6-T10: Generate the Premium Teacher Visual. P6-T11: Save the final Teacher Visual. Additional Premium Visuals P6-T12: Generate the Visual Production Script in Gemini. P6-T17: Save the final Premium Visuals. Phase 7 — Final Deliverables P7-T01: Verify the Lesson Plan is complete. P7-T02: Verify the Teacher Mind Map is complete. P7-T07: Verify the Instructional Infographic is complete. P7-T08: Verify the Presentation is complete. P7-T09: Verify the Revision Kit is complete. P7-T10: Verify the Classroom Activities are complete. P7-T11: Verify the Premium Formula Visual is complete. P7-T12: Verify the Premium Teacher Visual is complete. P7-T13: Verify the Additional Premium Visuals are complete. Phase 8 — Quality Assurance (QA) P8-T01: Verify all files are saved. P8-T02: Verify the naming convention is followed. P8-T03: Verify visual consistency across all assets. P8-T04: Check that no required resource is missing. P8-T05: Store everything in the project folder. Phase 9 — Classroom Delivery P9-T01: Introduce the chapter using the Premium Visuals. P9-T02: Teach using the Lesson Plan. P9-T03: Present the Slide Deck. P9-T04: Conduct the Classroom Activities. P9-T05: Conduct Revision using the Flash Cards. P9-T06: Conduct the Classroom Assessment Quiz. P9-T07: Share the Revision Kit. P9-T08: Collect student feedback. Phase 10 — Reuse the System P10-T01: Duplicate the project. P10-T02: Replace the source chapter. P10-T03: Restart the workflow from P1-T01.
