How to Use The Prompt

চ্যাটজিপিটি ব্যবহার করা কোনো কঠিন কাজ নয়। নিচের ঘরগুলোতে আপনার প্রম্পটের নাম এবং আপনার নির্দিষ্ট বিষয়টি লিখুন। আপনার জন্য প্রয়োজনীয় গাইডলাইনটি নিচে রিয়েল-টাইমে আপডেট হয়ে যাবে। কাজ শেষ হলে জাস্ট 'কপি' করে নিন।
