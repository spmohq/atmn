PGP Process

আপনার প্রোডাক্টিভিটি এবং আউটপুটকে পরবর্তী স্তরে নিয়ে যাওয়ার রোডম্যাপ

আপনি যদি নিজের প্রম্পটগুলো দিয়ে একটি সলিড প্রোডাক্ট তৈরি করে সেটি সেল করতে চান, তবে আপনি একদম সঠিক জায়গায় এসেছেন। নিচে আপনার কাজের সুবিধার জন্য ৩টি মাস্টার প্রম্পট সাজিয়ে দেওয়া হলো। আপনি জাস্ট কপি করে ChatGPT-তে ব্যবহার করতে পারবেন।

তার আগে তুমি আমাকে সহজ বাংলা ভাষাতে বুঝিয়ে দাও - 
এই কাজটা করার সরল পদ্ধতিটা কি (একটা minute detail-ও skip কোরো না বা কোনো step-কে jump কোরো না)?

মনে রাখবে যে, ইউজার হিসেবে আমি একজন Product Builder and Seller. 
আমার উদ্দেশ্য - 
১, Prompt Extraction, 
2, Packaging the Prompts as a Product, 
3, Writing its Sales Copy, 
4, generating its complete 100% mobile responsive html sales page (all codes, css, and scripts in a single html file) 
5, Getting A social media post, a Blig post, a newsletter, and a solo podcast with a call to action to send visitors to the sales page. All these four contents must be based on a single pshycological pain problem related to the product

PROMPT 2 আউটপুট নিশ্চিত করা

প্রতিটি স্টেপের শেষে আপনি ঠিক কী ফাইল বা রেজাল্ট পাচ্ছেন, তা পরিষ্কার করার জন্য এই প্রম্পটটি ব্যবহার করুন।

Master Prompt #2 
আমি প্রতিটি স্টেপের জন্য একটি ছোট Deliverable Prompt ব্যবহার করতে চাই, যেখানে AI-কে শুধু একটাই প্রশ্ন করা হবে:

"এই স্টেপের শেষে আমি কোন ফাইল বা আউটপুট হাতে পাব?"

যদি সেই প্রশ্নের উত্তর পরিষ্কার না হয়, তাহলে সেই স্টেপ অপ্রয়োজনীয়। 

- প্রতিটি স্টেপের জন্য বাংলাতে একটি ছোট Deliverable Prompt লিখে দাও - give them in codebox in markdown
