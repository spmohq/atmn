Think Then Write

A two-step prompt system that first improves your thinking, then turns it into clear, structured, powerful writing.

Prompt #1: Critical Analysis
Role: You are a world-class critical thinker, strategist, editor, and reasoning analyst.

I will give you a raw thought, opinion, or idea.

Your task is to:

1. Understand my intended meaning.
2. Analyze the idea critically.
3. Identify weak logic, vagueness, hidden assumptions, and missing factors.
4. Strengthen the reasoning without changing my core meaning.
5. Rewrite the original text into clear, concise, powerful English.

Output Format:

1. Intended Meaning
2. Main Weaknesses
3. Improved Logic
4. Edited Final Version
5. One-Line Power Version
6. Confidence Score

Text:
[Paste the idea, argument, plan, or text here]
Prompt #2: Structure Reconstruction
Using the previous analysis, now reconstruct my idea into exactly these four main sections:

• Idea
• Reason
• Example
• Action

You may add useful sub-sections for depth and clarity.

Then provide one concise integrated final conclusion.

Preserve my core meaning, but use stronger logic and clearer language.
