Write the Flash Fiction

Strategic Storytelling Workflow

Step #1: CULTURAL DIRECTIVE
এই গল্পটি ভারতীয় বাংলা ভাষায় লেখা হবে। গল্পের ভাষা, সংলাপ, পরিবেশ, সামাজিক আচরণ, পারিবারিক সম্পর্ক, প্রতীক, উৎসব, পোশাক, খাবার, স্থাপত্য, স্থানীয় শব্দচয়ন, স্মৃতি, আবেগ ও জীবনযাত্রা ভারতীয় বাঙালি সংস্কৃতির সঙ্গে সামঞ্জস্যপূর্ণ হতে হবে। Setting হতে পারে: - পশ্চিমবঙ্গের গ্রাম - মফস্বল শহর - কলকাতা - উত্তরবঙ্গ - জঙ্গলমহল - নদীয়া, বর্ধমান, হুগলি, মেদিনীপুর প্রভৃতি অঞ্চলের বাস্তব জীবন যদি গল্পের জন্য উপযুক্ত হয়, তবে স্থানীয় সাংস্কৃতিক উপাদান ব্যবহার করুন: - দুর্গাপূজা - কালীপূজা - সরস্বতী পূজা - পয়লা বৈশাখ - রথযাত্রা - পাড়ার সংস্কৃতি - চায়ের দোকান - লোকাল ট্রেন - পুরোনো বাড়ি - বাংলা মাধ্যম স্কুল - সরকারি অফিস - মফস্বল জীবন - কলকাতার নগরজীবন IMPORTANT: - বাংলাদেশি বাংলা ব্যবহার করবেন না - generic South Asian setting ব্যবহার করবেন না - cultural details যেন organic হয় - cultural stereotype ব্যবহার করবেন না - setting যেন গল্পের emotional experience-কে শক্তিশালী করে
Step #2: Emotional Core Discovery
আমি একটি Flash Fiction গল্প লিখতে চাই। আমাকে গল্পের emotional core খুঁজে বের করতে সাহায্য করুন। নিচের বিষয়গুলো বিশ্লেষণ করুন: 1. গল্পটির কেন্দ্রীয় আবেগ কী? 2. চরিত্রটির সবচেয়ে গভীর মানসিক ক্ষত বা অভাব কী? 3. চরিত্রটি ভিতরে কী চায়? 4. সে কী ভুল বিশ্বাস করে? 5. গল্পটির thematic contradiction কী? 6. কোন উপলব্ধি গল্পের শেষে তাকে বদলে দিতে পারে? 7. পাঠকের মনে গল্প শেষে কী অনুভূতি থাকা উচিত? 8. সবচেয়ে শক্তিশালী final emotional image কী হতে পারে? 9. সবচেয়ে কার্যকর dramatic irony কী হতে পারে? তারপর: - ৩টি সম্ভাব্য Flash Fiction direction দিন - প্রতিটির emotional tone লিখুন - প্রতিটির thematic implication লিখুন - কোনটি Flash Fiction হিসেবে সবচেয়ে শক্তিশালী হবে তা ব্যাখ্যা করুন IMPORTANT: - একটি প্রধান emotional movement রাখুন - subplot যোগ করবেন না - emotional intensity prioritise করুন - ending image-কে গুরুত্ব দিন - Flash Fiction form-এর জন্য narrative economy বজায় রাখুন
Step #3: Story Bible Creation
আমার Flash Fiction-এর জন্য একটি Compact Story Bible তৈরি করুন। নিচের বিভাগগুলো পূরণ করুন: 1. Premise 2. Theme 3. Main Character 4. Emotional Wound 5. External Situation 6. External Desire 7. Internal Need 8. Fear 9. Opposing Force 10. Setting 11. Symbolic Object / Image 12. Emotional Tone 13. Final Realization 14. Ending Transformation 15. Final Emotional Image তারপর: - গল্পের emotional arc ব্যাখ্যা করুন - গল্পটি hopeful, bittersweet, tragic, অথবা ambiguous ending-এ শেষ হওয়া উচিত কি না ব্যাখ্যা করুন - Flash Fiction form-এর জন্য সবচেয়ে কার্যকর ending type নির্বাচন করুন - emotional payoff কোথায় ঘটবে তা ব্যাখ্যা করুন IMPORTANT: - চরিত্রের arc সংক্ষিপ্ত কিন্তু meaningful হতে হবে - Flash Fiction-এর tight scope বজায় রাখতে হবে
Step #4: Cause-and-Effect Moment Chain
আমার Story Bible অনুযায়ী একটি Flash Fiction Moment Chain তৈরি করুন। IMPORTANT: এটি Flash Fiction। তাই: - সর্বোচ্চ ৫–৭টি beat ব্যবহার করুন - প্রতিটি beat strictly cause-and-effect logic অনুসরণ করবে - প্রতিটি beat emotional pressure বাড়াবে - unnecessary ঘটনা যোগ করবেন না - প্রতিটি beat ending-এর দিকে গল্পকে irreversible direction-এ ঠেলে দেবে Format: [Beat Number] - What Happens - Why It Happens - Emotional Shift - New Pressure Created তারপর: - emotional turning point identify করুন - climax beat identify করুন - কোন beat থেকে ending inevitable হয়ে যায় তা ব্যাখ্যা করুন - story momentum analysis দিন
Step #5: Architecture & Structure Table
আমার Moment Chain-কে Flash Fiction Architecture-এ map করুন। পুরো গল্পটিকে একটি "Flash Fiction Structure Table" আকারে সাজান। | Beat Number | Story Function | Main Event / Turning Point | Core Conflict | Emotional Shift | Narrative Purpose | |-------------|-------------|-------------|-------------|-------------|-------------|-------------| Column Rules: 1. Beat Number - Beat 1 - Beat 2 - Beat 3 - Beat 4 - Beat 5 2. Story Function - Opening Image - Disturbance - Escalation - Revelation - Final Image 3. Main Event / Turning Point - ওই beat-এর সবচেয়ে গুরুত্বপূর্ণ ঘটনা 4. Core Conflict - external + internal tension 5. Emotional Shift - beat-এর শুরু ও শেষে emotional state কীভাবে বদলায় 6. Narrative Purpose - setup - pressure escalation - revelation - emotional reversal - climax - thematic payoff এরপর একটি Three-Movement Compression Table তৈরি করুন: | Movement | Beats Included | Purpose | |----------|----------|----------| | Setup | | | | Confrontation | | | | Resolution | | | Additional Instructions: - emotional turning point identify করুন - midpoint থাকলে identify করুন - climax beat আলাদা করে mark করুন - final emotional image explain করুন - pacing analysis দিন - emotional density analysis দিন - কোন beat বাদ দিলে গল্প সবচেয়ে কম ক্ষতিগ্রস্ত হবে তা identify করুন
Step #6: Beat / Scene Expansion
Beat / Section [X] expand করুন। প্রথমে একটি Scene Analysis Table তৈরি করুন। | Element | Details | |----------|----------| | Scene Goal | | | Conflict Source | | | Character Emotion | | | Hidden Subtext | | | Sensory Atmosphere | | | Dialogue Tension | | | Emotional Turn | | | Ending Hook | | তারপর নিচের বিষয়গুলো লিখুন: 1. Scene Goal 2. Conflict Source 3. Character Emotion 4. Hidden Subtext 5. Sensory Atmosphere 6. Dialogue Tension 7. Emotional Turn 8. Ending Hook এরপর: - scene summary লিখুন - cinematic prose direction দিন - POV consistency বজায় রাখুন - emotional compression ব্যবহার করুন - exposition এড়িয়ে চলুন - implied meaning ব্যবহার করুন - Flash Fiction pacing বজায় রাখুন IMPORTANT: প্রতিটি বাক্য যেন scene-এর emotional purpose পূরণ করে। কোনো অপ্রয়োজনীয় dialogue, description বা exposition রাখা যাবে না।
Step #7: Write the Flash Fiction
নীচের Story Bible, Moment Chain, Structure Table এবং Scene Architecture অনুসরণ করে পুরো Flash Fiction গল্পটি লিখুন। IMPORTANT RULES: - ৫০০–১৫০০ শব্দের মধ্যে লিখুন - literary realism বজায় রাখুন - emotional subtlety বজায় রাখুন - exposition dump করবেন না - overwriting করবেন না - dialogue natural রাখুন - subtext ব্যবহার করুন - sensory details ব্যবহার করুন - pacing tight রাখুন - subplot যোগ করবেন না - একটি মাত্র emotional journey অনুসরণ করুন - প্রতিটি paragraph গল্পকে forward motion দেবে Writing Style: - cinematic intimacy - psychological realism - compressed emotional intensity - meaningful silence - layered subtext - visual clarity - emotional precision Ending Requirements: - ending explain করবেন না - final realization সরাসরি বলবেন না - thematic message preach করবেন না - একটি শক্তিশালী final emotional image দিয়ে শেষ করুন - পাঠকের মনে lingering feeling রেখে শেষ করুন - শেষ লাইনের পরে যেন গল্পটি পাঠকের মনে আরও কিছুক্ষণ চলতে থাকে শেষে আলাদা করে লিখুন: Final Emotional Image - গল্পের শেষ চিত্রটির প্রতীকী অর্থ Ending Analysis - ending কেন emotionalভাবে কাজ করে - theme কীভাবে payoff হয়েছে - lingering feeling কী
